# Album Release Exercise

## Português
Exercício prático desenvolvido para aplicar os conceitos iniciais de
Programação Orientada a Objetos em Java.

O programa recebe o nome de uma artista e um ano, retornando o nome de 
um álbum musical lançado ou se não houve lançamentos naquele ano.

### Conceitos aplicados
- Classes e objetos;
- Construtores;
- Métodos;
- Encapsulamento básico;
- Sobrescrita de `toString()`.

Esse projeto faz parte dos meus estudos iniciais em Java e OOP.

---

## English
Practical exercise developed to apply the first concepts of
Object-Oriented Programming in Java.

The program receives an artist name and a year, returning the name of an album
released or if there is not a released album in this following year.

### Concepts covered
- Classes and objects;
- Constructors;
- Methods;
- Basic encapsulation;
- `toString()` method override.

This project is part of my initial studies in Java and OOP.
